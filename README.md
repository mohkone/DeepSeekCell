**DeepSeekCell** is an AI‑powered cell type annotation tool that leverages large language models (LLMs) and the Cell Ontology to provide accurate, interpretable annotations for single‑cell RNA‑seq clusters. This repository accompanies the publication:

> *DeepSeekCell: Benchmarking Large Language Model‑Powered Cell Type Annotation with Ontology‑Aware Evaluation and an Interactive Shiny Application.*  
> *Computers in Biology and Medicine*, 2026.
